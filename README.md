## 🎯 Strategie-Säulen

- **Bootstrap via Community** — ps-connect Bestandsnutzer als kostengünstige Erstnutzer
- **Virales Wachstum** — Referral-Programm + Gamification (Viral Coefficient Ziel: 0,30)
- **B2B Upsell** — Club-Lizenzen & Verbands-Partnerships (DFB, FFF, RFEF, FIGC)
- **Paid Scale** — Influencer + Social Ads + Content Marketing ab Phase 2
