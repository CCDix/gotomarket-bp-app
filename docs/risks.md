## Contingency-Playbooks

### 🚨 Szenario A: Langsame Community Adoption
**Trigger:** < 800 Signups in Woche 1–2 (statt 1.500)

```
Sofortmaßnahmen:
1. Emergency €20K Paid Ads starten (Instagram + TikTok)
2. CAC erhöht sich auf ~€10 → akzeptabel bei LTV €135
3. Influencer-Posting-Frequenz erhöhen (2x/Woche → 3x/Woche)
4. Persönliche Outreach-Kampagne an Top-ps-connect-User
```

### 🚨 Szenario B: Influencer-Kampagne floppt
**Trigger:** < 50K Impressions nach 4 Wochen (statt 120K+)

```
Sofortmaßnahmen:
1. Influencer-Budget (€5K) zu Content Marketing umlenken
2. 5 neue Influencer kurzfristig onboarden
3. Paid Boosting für organische Posts (€3–5K)
4. User-Generated Content Kampagne starten
```

### 🚨 Szenario C: Hoher Churn (> 5%)
**Trigger:** Monthly Churn > 5% nach ersten 60 Tagen

```
Sofortmaßnahmen:
1. Churn-Interviews mit abgewanderten Nutzern (10–20 Interviews)
2. Onboarding-Flow verbessern (Activation-Rate erhöhen)
3. Feature-Entwicklung beschleunigen (Roadmap priorisieren)
4. Re-Engagement-Kampagne: E-Mail + Push-Notification
```

### 🚨 Szenario D: KI-Genauigkeit unzureichend
**Trigger:** Beta-Feedback zeigt < 80% Zufriedenheit mit KI-Analyse

```
Sofortmaßnahmen:
1. Launch verschieben bis Quality-Gate erfüllt
2. Manuelle Review-Schicht als Qualitätssicherung einbauen
3. KI-Modell nachtrainieren (erweiterte Beta-Daten)
4. Transparente Kommunikation: "Beta-Phase" klar labeln
```
